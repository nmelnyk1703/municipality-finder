# Municipality Finder

Given a **geolocation (lat/long)**, a **street address**, or **both**, this tool
returns the U.S. **municipality** (city / village / town) the location belongs to.

If a second municipality lies within a **3-mile radius**, both are reported, each
with a **confidence rating** for which one the location most likely belongs to.

> Built in Python (best fit for a geocoding + LLM tool). The `pom.xml` / `Main.java`
> are the original empty IntelliJ Java stubs and are unused.

## How it works

1. **Resolve to coordinates** — an address is forward-geocoded; coordinates are used directly.
2. **Find the municipality** — the point is reverse-geocoded against the free
   [U.S. Census Geocoder](https://geocoding.geo.census.gov/). The **County Subdivisions**
   layer is authoritative: in Wisconsin, cities, villages, and towns are all Minor
   Civil Divisions there.
3. **3-mile scan** — sample points are cast on rings (1/2/3 mi) around the location;
   any other municipality found within 3 miles is reported as a neighbor.
4. **Confidence** — the point sits inside exactly one boundary, so that municipality
   gets the highest confidence. Neighbors get modest confidence that grows the closer
   their boundary is (accounting for slightly-off input). Reasoning is generated by the
   **Claude API** when `ANTHROPIC_API_KEY` is set, otherwise by a deterministic
   distance heuristic.

## Requirements

- Python 3.10+ (uses only the standard library).
- Optional: `ANTHROPIC_API_KEY` env var + `anthropic` package for AI-generated
  confidence reasoning. Without it, a built-in heuristic is used.

## Usage

```bash
# By address
python3 municipality_finder.py --address "800 Algoma Blvd, Oshkosh, WI 54901"

# By coordinates
python3 municipality_finder.py --lat 43.0731 --lon -89.4012
python3 municipality_finder.py --coords "43.0731,-89.4012"

# Both (coordinates take precedence for locating)
python3 municipality_finder.py --address "..." --lat 43.07 --lon -89.40

# Machine-readable output
python3 municipality_finder.py --coords "43.0731,-89.4012" --json
```

### Example

```
$ python3 municipality_finder.py --lat 43.0722 --lon -89.5089

  → Madison city   [Dane County, WI]    confidence 70%

Other municipalities within 3 miles:
  • Middleton city  (~1.0 mi)   confidence 20%
  • Middleton town  (~2.0 mi)   confidence 10%
```

## Notes / limits

- **Coverage:** United States (Census data). Best-tested for Wisconsin.
- Neighbor distances are *approximate* (nearest sampled point inside that
  municipality) — increase `BEARINGS` / `RING_RADII_MILES` in the script for finer
  resolution at the cost of more API calls.
- Points in unincorporated areas outside any county subdivision return an error.
